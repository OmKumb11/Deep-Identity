# 🛡️ Deep-Identity: Neural Forensic Suite

A multi-modal deep learning prototype for verifying digital integrity using **Xception-Net** and **Error Level Analysis (ELA)**.

## 🚀 Features
- **Face DNA Extraction**: Automated localization using MTCNN.
- **Identity Audit**: Forensic classification via Xception (FaceForensics++ trained).
- **Scene Integrity**: Metadata-level ELA for detecting object manipulation.
- **XAI Heatmaps**: Grad-CAM visualization of facial artifacts.
